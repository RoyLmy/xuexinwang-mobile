# 学信网移动端应用

基于 [学信网官方网站](https://my.chsi.com.cn/) 设计的移动端Web应用，完整复现了学信档案的功能和界面。

## 📱 项目简介

这是一个高仿学信网移动端的Web应用，适配iPhone 15屏幕尺寸，提供完整的学籍学历查询功能。项目基于纯HTML/CSS/JavaScript开发，无需框架依赖。
